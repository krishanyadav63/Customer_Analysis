# Customer_Analysis
Customer Behavior Analysis project using SQL, Python, and Power BI. Analyzed customer purchase patterns, segmentation, and key metrics to understand buying behavior. Built interactive dashboards in Power BI to support data-driven business decisions.

# Data Analysis Project

## Overview
This project demonstrates end-to-end data analysis using Python, SQL (MySQL), and Power BI to generate meaningful business insights from raw data.

## Dataset
- CSV business dataset containing sales, customer, and product information.

## Tools
- Python (Pandas)
- MySQL (SQL queries)
- Power BI (Dashboard)
- Jupyter Notebook

## Steps
- Loaded and cleaned data using Python  
- Performed Exploratory Data Analysis (EDA)  
- Ran SQL queries in MySQL  
- Built an interactive Power BI dashboard    

## Dashboard
Power BI dashboard shows sales trends, KPIs, and product/customer insights.

## Results
Identified key trends and patterns to support data-driven decision-making.

## How to Run
1. Run Python notebook for data cleaning and EDA  
2. Execute SQL queries in MySQL  
3. Open Power BI file to view dashboard
